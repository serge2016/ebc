1. MSM -> DAT using Mascot:
Порядок действий для обсчета доступных MSM-файлов (02.08.2016) на компьютере в 610.
1.1. Запускаем Mascot Daemon (Start -> Mascot Daemon). Идем во вкладку PArameter Editor. Open -> d:\MASCOT_DATA\Mascot_Daemon_Kristina04_NCBInr.par и проверяем параметры. При необходимости (если что-то требуется изменить) то сохраняем под НОВЫМ именем!
1.2. Идем во вкладку Task Editor. Parameter set: ... -> выбираем проверенный или созданный в прошлом пункте файл. Task: вводим называние задачи без пробелов.
1.3. Идем в Total Commander. Переходим в папку d:\MASCOT_DATA\from_Kristina\. Выполняем поиск *.msm файлов внутри этой папки (Alt+F7). Нажимаем "Файлы на панель". Выделяем их все (Ctrl+A) [288 файлов]. Если нужно, то снимаем выделение с "лишних" файлов. Перетаскиваем выделенные файлы мышкой в белое поле Data file list в Mascot Daemon. ПРИМЕЧАНИЕ! Такая процедура добавления задачи нужна, чтобы сохранить правильный алфавитный порядок образцов в том числе из разных датасетов. Или же разные наборы образцов нужно добавлять в правильном порядке в одну задачу. Иначе задачи будут выполняться параллельно и будет путаница в выходных файлах.
1.4. Проверяем наличие достаточного количества свободного места на диске C: [по 7 Мб на 1 DAT-файл, т.е. примерно 2,5-3 Гб].
1.5. Нажимаем Run. Ждем около суток.
1.6. По завершению задачи в папке c:\PF2\Mascot\data\ появится папка(-и) с датой выполнения задачи, внутри которой(-ых) будут *.dat-файлы. dat-файлы F00????.dat, которые можно сконвертировать в htm-reports с помощью п.2

2. DAT -> HTMs using Mascot script (этот шаг уже можно делать на любом линуксе):
dat-Файлы лежат в c:\PF2\Mascot\data\ в подпапках с датой обсчета.
Запустить скрипт Kristina_datToHtm_v05_1.cmd для конвертации mascot *.dat (результат п.1) в htm-файлы с белковой и htm-файлы с пептидной выдачами одновременно. При этом htm-файлы уже будут переименованы в соответствии с именами msm-файлов, сохраненными в dat-файлах.
ВАЖНО! в нем нужно поправить значения в строках:
	set inputDir=c:/PF2/Mascot/data/20181104
	set outputDirProts=c:\PF2\Mascot\Kristina\Mia2018\Mia2018_4htmProts
	set outputDirPepts=c:\PF2\Mascot\Kristina\Mia2018\Mia2018_4htmPepts

Префикс в браузере: http://filimonovpc/mascot/cgi/
Для корректной работы ссылок на компьютере в 610 во вручную сгенерированных htm-файлах в них внутрь тега <HEAD> надо дописать:
	<base href="http://filimonovpc/mascot/cgi/">
Сейчас это делает скрипт Kristina_datToHtm_v05_1.cmd.
